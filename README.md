# Deftia

Deftia는 1인칭, 신속한 진행이 중점인 FPS 게임입니다.
직관적인 움직임과 짜릿한 속도감을 목표로 개발 중입니다.

## 주요 기능

* **플레이어 움직임**: WASD 키를 사용한 이동
* **다양한 이동 기술**: 점프 및 슬라이드
* **1인칭 시점**: 마우스로 시점을 자유롭게 조작하는 FPS 시점

## 개발 환경

* **엔진**: Unity
* **언어**: C#
* **버전 관리**: Git & GitHub
